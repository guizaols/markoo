# MVP Scope (Free Track)

## Must-have
- Fast search, folder tree navigation, add/edit/delete, sort by name/date.
- Stable experience with 5,000+ bookmarks.
- Basic telemetry for funnel and performance.

## Out of scope for launch
- Billing and subscription flows.
- Team collaboration.
- AI summarization.

## Definition of done
- End-to-end flows covered by unit + e2e baseline.
- CI green on every PR.
- Privacy policy and terms published.
- Chrome Web Store listing assets ready.
