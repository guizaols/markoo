# Chrome Web Store Listing Kit

## One-line pitch
Markoo helps professionals find and organize large bookmark libraries in seconds.

## Short description
Fast bookmark search, folder-first navigation, and clean actions for high-volume bookmark workflows.

## Full description
Markoo is built for users with hundreds or thousands of bookmarks. Navigate deep folder trees, search instantly, sort quickly, and keep your links maintainable with focused, minimal controls.

### Highlights
- Search optimized for large bookmark sets.
- Folder/subfolder exploration with counts.
- Fast add/edit/delete actions.
- Enterprise-ready visual style and interaction model.

## Privacy summary
Markoo operates locally for bookmark management. If telemetry is enabled, only product usage metrics are collected; bookmark content is not transmitted by default.
